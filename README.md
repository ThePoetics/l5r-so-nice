# L5R-so-Nice

A collection of Clan-based die presets for use with the [Foundry virtual table top software](https://foundryvtt.com), the [Dice so Nice](https://foundryvtt.com/packages/dice-so-nice/) visual plugin module, and the *Legend of the Five Rings* game system (either [4th Edition](https://foundryvtt.com/packages/l5r4) or [5th Edition](https://foundryvtt.com/packages/l5r5e)).
